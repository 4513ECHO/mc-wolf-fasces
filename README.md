# Fasces

Minecraft datapack for controlling tamed wolves.

## Usage

- To get the fasces: `/function fasces:create_fasces`
- <kbd>Right Click</kbd>: Toggle wolves sitting.
- <kbd>Shift</kbd>+<kbd>Right Click</kbd>: Teleport wolves to the player.
- On multiplay, the owner of wolves changes every time the player use the fasces.
